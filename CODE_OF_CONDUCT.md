# Código de Conducta / Code of Conduct

## Nuestro compromiso

En el proyecto Forja, nos comprometemos a mantener un entorno abierto, acogedor y seguro para todas las personas, independientemente de:

- Nivel de experiencia
- Género, identidad y orientación sexual
- Discapacidad
- Apariencia física
- Etnia, nacionalidad
- Edad
- Religión u otras creencias
- Orientación técnica (lenguajes, métodos, etc.)

## Comportamiento esperado

- Ser acogedor con opiniones y críticas constructivas
- Respetar el tiempo y esfuerzo de otros colaboradores
- Documentar código con claridad y ayudar a otros a entenderlo
- Reconocer las contribuciones de otros manteniendo el crédito

## Comportamiento inaceptable

- Comentarios despectivos, insultos o ataques personales
- Harassment público o privado
- Publicar información privada sin permiso
- Otras conductas deshonestas o inapropiadas

## Reportar problemas

Contactar a lococoi a través de GitHub o correo para reportar violaciones.
